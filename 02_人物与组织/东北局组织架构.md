# 东北局组织架构

> 本文档以《人物小传_新增角色.md》为新增人物唯一来源。当前同人新增角色共有八人：安楠、邓敏、赵大全、林剑行、易冉武、范云淮、柳晴川、旗铜委。其中柳晴川为非常驻铁道顾问，其他已出现姓名作为原作人物、上级机关接口、背景力量或岗位参照处理，不再占用可替换岗位。

---

## 一、安排原则

1. **新增角色优先落岗**：能由八名新增角色承担的关键岗位，优先替换原先的“待定”“真空”或原作人物占位。
2. **不强行改写上级机关**：马千瞩、鹿文渊、程栋、明朗、江山、郭逸、陈海阳等仍作为原作权力接口存在，但不是本同人新增岗位主体。
3. **一人一主岗，可兼接口**：每名新增角色必须有一个清晰主岗；兼任职责只用于制造组织张力，不让职责散掉。
4. **真空岗位可保留**：若八名角色不适合某岗位，不硬塞。保留真空比让人物失真更好。

## 三、顶层架构

```
元老院本部 / 主席团
    │
    ├── 政务院                         ← 行政归口
    ├── 联合参谋本部                   ← 军事指挥归口
    ├── 海军部                         ← 港区军令归口
    ├── 对外情报局                     ← 外线情报、朝鲜方向
    ├── 政保总局                       ← 内部安全、涉密审查
    ├── 财政部 / 中央银行 / 审计署      ← 账户、债券、监察
    │
    └── 辽南特别军事开发管制区
            │
            ├── 004 号执行筹备组（过渡期）
            └── 辽南军政委员会（成熟期）
```

**辽南特别军事开发管制区**
- 权属归口：政务院体系
- 军港优先：海军军令系统直接签字
- 经营工具：北方开发公司
- 监督工具：财政、审计、政保、牺牲审查、情报归口分线制

> 北方开发公司只是经营执行工具，不能代替政务院、军令机关、司法机关和监察机关行使最终公权。

## 六、情报与政保双线

> 外线情报归对外情报局；内部安全、涉密人员、政治档案归政保总局。两线不合并，但必须有一个能签字、能否决、能封存争议的协调点。

```
对外情报局
    └── 北方站 / 北方情报协调处
            └── ★ 赵大全：实地协调员
                    ├── 朝鲜义州联络网实际管理
                    ├── 联络员培养与接触行动设计
                    └── 公开身份挂黄骅贸易结构

政保总局
    └── 北方安全监察组
            └── ★ 旗铜委：派驻代表兼双线协调官
                    ├── 审查赵大全拟用朝鲜联络员
                    ├── 监控旅顺内部信息暴露面
                    ├── 有权否决特定联络员使用资格
                    └── 与赵大全形成结构性张力
```

**岗位替换结果**
- 原“政保/情报双线派驻协调岗”由 **旗铜委** 填补。
- 原“外线实地协调”由 **赵大全** 明确主责。
- 两人互不统属：赵大全管外部接触，旗铜委管内部安全与资格审查。
