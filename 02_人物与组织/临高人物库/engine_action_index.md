# 发动机行动·人物专项索引

## 发动机行动·人物专项索引

> 发动机行动是元老院赌国运级别的山东登莱攻略，同期包含济州岛（耽罗）D日攻占行动。前指解散后核心成员多被边缘化，是鹿文渊北上联盟的重要情感与政治基础。
> 相关归化民小传详见 `../人物小传_新增归化民.md`。
