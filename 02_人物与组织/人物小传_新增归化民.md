# 旅顺口同人 · 新增归化民人物小传

> 本文件收录49章以后旅顺口线新增或明确规划复用的归化民角色。八名新增元老另见 `人物小传_新增角色.md`。  
> 标注为“原著续用”的人物，不改原著既定事实；标注为“同人新增/暂名”的人物，姓名可按正文需要替换，核心岗位和戏剧功能先固定。

---

## 使用原则

- 归化民不是岗位标签。每个可复用人物至少要有来历、能力、弱点、关系入口和后续戏份。
- 暂名角色先服务剧情结构：落地、旧医登记、铁路教育、工业安全、牺牲审查、海关账务。名字可换，功能不要轻易换。
- 群体条目只用于暂时不需要拆成单人的人群；一旦在正文中承担选择、冲突或牺牲，就应拆出独立小传。

## 一、原著北方线关键归化民

### 毛俊 / 毛十三

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 原名 | 毛十三；鹿文渊曾先改作“毛君”，后改为毛俊 |
| 身份 | 东江镇军户出身；薛子良海天号北方侦察时救起；后受对外情报局训练 |
| 现状 | 黄骅旧部，后转入鹿文渊身边任归化民秘书、通讯与北方事务助手 |
| 核心定位 | 从“被北方线救起的人”变成“带着北方线回到旅顺的人” |
| 关系入口 | 薛子良、黄骅、鹿文渊、屺坶岛商务处、镇江堡旧路、东江军户线 |
| 旅顺功能 | 登陆前视角、旧侦察报告回收、东江军户记忆、北方线文书与通讯执行 |

毛俊最重要的价值，不是“懂辽东”的工具属性，而是他本人的来路正好压在旅顺线的根上。原著第一百二十七节至一百二十九节里，薛子良的海天号侦察山东至旅顺、鸭绿江口和镇江堡一线，毛十三作为镇江堡附近的东江军户少年，在冰滩上因饥寒几乎倒毙，被侦察队救上船。此后他跟随黄骅，接受对外情报局训练，掌握普通话和报务技能，后来被鹿文渊要到屺坶岛身边。

旅顺口线中，毛俊应作为原著北方侦察线的回流人物使用。他不是被黑烟铁船吓跑的本地人，而是曾经亲眼看过东江军户饥饿、镇江堡破败、鸭绿江边荒岛白骨的人。登陆旅顺前，他可以重读薛子良旧侦察记录，也可以随鹿文渊处理屺坶岛、镇江堡、旅顺之间的文书、通讯和人员转运。这样写能把“旅顺不是新地图，而是旧北方线的汇合点”落到一个具体归化民身上。

毛俊的弱点也应保留。他年轻，仍有旧军户出身留下的畏惧和服从习惯；他称鹿文渊为“老爷”的旧称呼不宜一下改干净。正因为如此，他很适合承担登陆前的内心视角：他知道自己已经不是当年那个饿昏在冰滩上的孩子，但再次看见旅顺、镇江、东江这些地名时，身体记忆会比政治判断更早反应。

后续戏份可集中在三处：一是登陆前整理旧侦察材料，作为“旧海路”章节的主视角；二是旅顺初到后协助旗铜委、黄骅核对镇江堡旧路和接头人失联；三是鹿文渊身边归化民秘书线，负责把元老的战略命令翻译成现场能执行的表格、口信和电报。

## 二、发动机行动旧部与山东线

### 黄安德

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 伏波军归化民军官，发动机行动山东线重要参与者 |
| 核心定位 | 危城夺宝行动中发挥不可替代作用的归化民旧部核心 |
| 关系入口 | 朱四、黄熊、王七索、鹿文渊、朱鸣夏、魏爱文总政治部档案 |
| 旅顺功能 | 旧部召集、归化民军官训练、北方实战经验、伤退牺牲待遇线 |

黄安德的价值不在某项技术，而在“登州城里把命交出去过”。他不是被动跟随元老的普通归化民，而是在发动机行动中证明过胆量、判断和组织力的人。原著后段“三百七十一节 聚会（一）”至“三百七十五节 特别任务”里，他因危城夺宝功劳获得军官住房指标，并能把登州旧部、警备营士兵和黄熊这样的边缘旧人聚到一张桌上。

旅顺口线中，黄安德应作为发动机行动旧部的归化民代表使用。他可以被朱鸣夏或魏爱文的名册重新调出，也可以被鹿文渊借来做旧部安抚、训练和人员核查。注意：他不属于医学院、卫生行政或儿童营养线。

### 朱四

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 原伪明登莱镇兵，朱三弟弟；发动机行动参与者 |
| 现状 | 临高警备营服役，已与郑月娥成婚 |
| 核心定位 | 最贴近普通旧兵生活的发动机行动归化民 |
| 旅顺功能 | 警备营接口、基层军纪视角、旧部伤退待遇线 |

朱四不适合写成高谈阔论的人。他的戏应该落在具体生活上：营房、休假、妻子、上级命令、旧日战友。他知道旧军队是什么样，也知道澳宋军纪和旧军队之间的差别，但他未必能把这套东西说成理论。

旅顺口若调用朱四，最适合放在警备、哨卡、临时营地和旧部安置场景。他能让“发动机行动旧部”不只停留在元老之间的情义，而落到一个士兵的日常和风险上。

### 许哲伟

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 山东莱州账房，登莱之乱幸存者 |
| 现状 | 经济口归化民干部 |
| 核心定位 | 登莱旧事与山东账务的活档案 |
| 旅顺功能 | 旧账复核、地方财务、招远矿账、山东线证言 |

许哲伟的底色是“账房”和“幸存者”。孔军叛乱时全家遇难，使他对失控军队、乱账和前线承诺有天然警惕。他不是战斗型人物，但在旅顺口这种账、粮、工、军、民混在一起的地方，账房比刀更能揭露问题。

后续可让他参与招远矿区账目、北方债券抵押、旧部待遇和山东逃民登记。他与邓敏的牺牲审查线天然有交叉：一个问账怎么挂，一个问人怎么死。

### 黄熊

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 北直隶河间人，早期归化民；黄安德堂兄 |
| 经历 | 老保安团资历，参加发动机行动后因处分影响晋升，曾在济州岛白马队任教官 |
| 核心定位 | 有资历但长期边缘的旧部样本 |
| 旅顺功能 | 旧部不满、军官住房、家属安置、再培训晋升线 |

黄熊代表另一类旧部：不是没有功劳，而是功劳、处分、资历和机会错位。他能理解黄安德的成功，也难免觉得自己被耽误。这个人物不宜写成反派，他的牢骚来自真实生活压力：妻子、孩子、住房、晋升和旧资历被新制度重新估分。

旅顺口使用黄熊时，可以把他放在军官再培训、白马队经验、骑兵或民兵训练上。他身上最适合写“元老院制度的公平”和“旧人情的期待”之间的摩擦。

### 王七索

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 警备营士兵，黄安德聚会线参与者 |
| 核心定位 | 从黄安德聚会线接入“特别任务”的基层警备视角 |
| 旅顺功能 | 警备营抽调、文化祭/少首长警卫经验、内卫纪律样本 |

王七索不是发动机行动核心旧部，但他在黄安德最后一组正文戏份里承接了“特别任务”线。他适合写成警备系统中的基层聪明人：表面喝酒说笑，实际知道自己什么时候该清醒、该上报、该自告奋勇。

如果旅顺口后续需要一个警备营旧人被抽调北上，王七索比纯新角色更有承接感。他能把临高内卫经验带到旅顺，但也会遇到更复杂的前线环境。

## 三、济州岛后院与发动机行动余脉

### 金六顺

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 济州岛朝鲜官奴婢出身，D日后归化 |
| 关系 | 金五顺之弟 |
| 核心定位 | 济州归化民干部体系的基层样本 |
| 旅顺功能 | 济州人口转运、净化营经验、朝鲜人群沟通 |

金六顺适合承担“从被征发者到执行者”的转变。他懂官奴婢出身的人怕什么，也懂元老院的登记、配给和纪律如何改变一个人的命运。这样的人在旅顺净化营、朝鲜逃民和济州转运人口之间很有用。

写他时不宜过度拔高。他的能力应表现为：知道谁在说谎，知道谁只是害怕，知道一条命令传到最底层会变成什么样。

### 金五顺

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用，原条目较少 |
| 身份 | 济州岛官奴婢出身，金六顺之姐 |
| 核心定位 | 济州女性归化民与家属安置视角 |
| 旅顺功能 | 女性净化营、工坊分流、家属就业、朝鲜妇女沟通 |

金五顺可以作为济州线的女性视角补足金六顺。她未必是正式干部，但可以成为实际执行层：分饭、点名、带队、调解女工矛盾、判断女人和孩子是不是被迫说谎。

她的戏剧功能是把净化营和家属安置写具体。旅顺口不是只有军官和工匠，真正稳定下来的标志是女人和孩子不再只是“随员”。

### 金勇柱

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 济州岛白丁猎户，D日后归化 |
| 核心定位 | 山野、猎路、边地直觉 |
| 旅顺功能 | 向导、猎户、马场/山林管理、逃人追踪 |

金勇柱的能力来自身体和地形。他不是文书型干部，但在济州、辽南、朝鲜北道这种山地、海岸、边界交错的空间里，能看出文书看不见的东西：脚印、烟火、猎道、避风处、牲畜异常。

旅顺口使用他时，可以让他成为“边地经验”的入口。他未必完全理解元老院的大计划，但他会判断一条路能不能走、一个人是不是刚从山里出来。

### 李万姬

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 济州岛归化民，政治保卫总局告密员 |
| 核心定位 | 政保系统在济州归化民中的眼线 |
| 旅顺功能 | 黑市、谣言、秘密会社、逃民群体侦听 |

李万姬不应写成脸谱化密探。她的告密可能来自恐惧、算计，也可能来自对旧秩序的厌恶。她知道底层人怎么传话、怎么藏东西、怎么用亲戚和同乡关系绕开登记。

旅顺口如果扩大净化营和朝鲜人口转运，李万姬可以作为政保系统的灰色工具。她有效，但她的存在会让安楠、邓敏和林剑行不舒服。

### 朴德欢

| 项目 | 内容 |
|------|------|
| 来源 | 原著续用 |
| 身份 | 济州岛归化民干部 |
| 核心定位 | 济州基层行政执行层 |
| 旅顺功能 | 配给、登记、劳工队管理、朝鲜语行政沟通 |

朴德欢适合写成“不出彩但离不开”的人。他负责把命令落到表格、口粮、点名和工棚上。这样的人不会在大会上发言，但没有他，所有漂亮规划都会在第二天早饭前乱套。

旅顺口中，他可以被冯宗泽或济州系统借调，协助管理来自济州、朝鲜和辽东的混合人群。

### 奉公队

| 项目 | 内容 |
|------|------|
| 来源 | 原著集体条目 |
| 身份 | 济州岛官奴婢及俘虏组成的劳工队 |
| 核心定位 | 济州劳动制度的集体缩影 |
| 旅顺功能 | 净化营劳务、道路修筑、转运体系、身份转化争议 |

奉公队目前先作为集体条目使用。它的价值是提醒旅顺口线：归化不是一句口号，劳工队、净化营、俘虏、官奴婢、战时征发之间存在灰色地带。

如果正文中出现具体选择、事故或牺牲，应从奉公队中拆出单独人物，不要长期只写“某队”“若干人”。

## 四、旅顺口规划新增归化民

### 金德寿

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增 |
| 身份 | 朝鲜北道矿工出身，兼译工，懂一点汉话 |
| 接入线 | 朝鲜矿产线、工业建设线、赵大全外线 |
| 核心定位 | 朝鲜矿产“有，但不好拿”的活证人 |

金德寿曾在朝鲜北部矿山服役，因战乱、税役或政治牵连逃到边境。他知道矿在哪里，也知道矿山背后的官吏、役夫、逃户和债务。他提供的不是现代地质报告，而是矿工记忆：哪条沟有绿石，哪座山雨后出黄水，哪处矿洞死过人。

他的矛盾很清楚：他不完全信任旅顺，也不愿回朝鲜；他说出矿点，可能救自己，也可能害死故乡的人。这个人物适合连接赵大全、邓敏、安楠和工业口。

### 侯铁良

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增 |
| 身份 | 辽东铁匠，后成为炉前班班头 |
| 接入线 | 铁山工业区、炉前班、归化民技术工 |
| 核心定位 | 传统经验与现代规程之间的桥 |

侯铁良看火色、听炉声、闻渣味。他的经验是真的，但只适合小炉、小作坊和熟悉的料。旅顺工业区要做的是连续生产、标准化配料和事故追责，这会让他一开始很不服气。

他的成长线不是“旧经验被打倒”，而是旧经验被规程吸收。他可以成为第一批真正意义上的归化民技术班组长，最后主持一炉“没有元老盯炉”的生产。

### 宋佩兰

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增 |
| 身份 | 化工所安全员，医务或护理训练出身的归化民助手 |
| 接入线 | 化工安全、工伤、女工岗位、邓敏审查线 |
| 核心定位 | 把工业事故从数字写成身体的人 |

宋佩兰负责的不是大理论，而是护目镜、围裙、冲洗水、烧伤记录、废液桶和澡堂。她知道酸厂不能停，也知道每一次“赶工”最后都会落在某个人的皮肤和眼睛上。

她与邓敏天然有交叉。邓敏要编号和责任链，宋佩兰知道伤口长什么样、谁先喊的救命、谁因为怕扣工分没有上报。

### 石明达（暂名）

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增，暂名可改 |
| 身份 | 负责旅顺先遣驻地的归化民头目 |
| 首次功能 | 第50章后接头人失联现场，协助鹿文渊、朱鸣夏处理驻地事务 |
| 核心定位 | “规划落地以后，谁在现场扛事”的执行层 |

石明达不是大人物，但他最早知道木桩打到哪里、粮袋放在哪间棚、哪个哨位昨夜换过人。他的权威来自“我在现场”，不是官衔。

他的弱点是容易把事压在自己手里，不愿立刻上报丢脸。接头人失联事件中，他可以既是帮手也是风险：他掌握一手细节，却可能因为怕担责延误判断。

### 刘应仓、马贵（暂名）

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增，暂名可改 |
| 身份 | 两名会辽东方言的归化民 |
| 接入线 | 接头人失联、旧医登记、辽东流民摸底 |
| 核心定位 | 翻译之外的“听口音、听来路、听破绽” |

这两人不要写成同一个功能。刘应仓可以偏沉默，擅长听口音和地名；马贵可以偏外向，擅长在茶棚、码头和难民堆里搭话。一个听，一个问。

他们的作用不只是翻译。他们能判断一个人是不是辽东本地、是不是山东逃来、是不是刻意学口音。赵大全和旗铜委都需要这样的人，但政保局也会盯着他们，因为他们接触的都是灰色消息。

### 周承业（暂名）

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增，暂名可改 |
| 身份 | 柳晴川留下的归化民铁道助教 |
| 接入线 | 旅顺口铁路与铁道教育线 |
| 核心定位 | 被骂到会看图纸的第一代铁路基层教师 |

周承业原本可以来自木匠、工兵队或港务脚夫，识字不多但手稳。他被柳晴川骂过无数次，最后不是变成工程师，而是能看懂坡度、排水、超载和道床为什么会杀人。

他的戏份适合承接柳晴川离开后的空档。第一次铁路小事故后，他不能再躲在“元老顾问不在”的借口后面，必须把传习班、巡检表和维修纪律撑起来。

沈文录不是战士，也不是法官。他的工作是问最难问的问题：谁最后见过他，谁下的命令，药给了没有，尸体在哪里，遗物谁收的。很多人会讨厌他，因为他让“已经过去了”的事过不去。

这个人物适合长期跟邓敏线。他可以让牺牲审查不只是元老口号，而变成一支笔、一只档案箱和一个会被前线所有部门嫌麻烦的归化民干部。

余秉诚的能力不是算术，而是知道账本怎么骗人。他在广州税务局见过士绅、买办、商人和衙门旧吏怎么把一笔货拆成三笔，把一艘船变成两张单。

魏长清表面是草棚里的郎中，会接骨、看冻伤、开几味常见药。他真正有用的地方不止医术，而是他治过谁、从哪条路来、哪些人不愿登记、哪些病号其实是逃兵或细作。

安楠想登记他的医案，赵大全想听他的来路，旗铜委想确认他有没有暗线。这个人物适合做旅顺落地后的第一场小型复杂戏。

### 罗小满（暂名）

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增，暂名可改 |
| 身份 | 净化营卫生员，北方医学专科筹备班第一批学员 |

| 核心定位 | “不是医生，但要先把人救到医生来”的前线卫生员 |

罗小满不需要写成天才。他只要会洗手、烧水、隔离、包扎、报病、数死人，就已经比大多数旧世界营地强。问题在于，前线会不断要求他做超出资格的事。

他的戏剧矛盾是岗位边界。安楠需要他救急，邓敏要保护他不替制度背锅，前线军官只看见“这里有个会包扎的人”。这个人物能把医学教育附款写具体。

### 六名北方医学培训生（群体）

| 项目 | 内容 |
|------|------|
| 来源 | 同人规划群体 |
| 身份 | 安楠带去旅顺的医学/卫生训练种子 |
| 接入线 | 医学专科筹备班、净化营、军队卫生员、旧医登记 |
| 拆分条件 | 任一人承担死亡、事故、恋爱、背叛、晋升等独立剧情时，应拆出个人小传 |

这六人暂时作为群体条目使用。建议构成不要全是同一类学生：至少包括一名芳草地基础好的学生、一名净化营出身识字者、一名军户或士兵出身卫生员、一名女护士预备生、一名旧医徒弟、一名药房/病案方向学生。

他们的共同主题是“半成品”。北方第一年等不到完整医生，只能让半成品先顶上去。戏份重点应放在培训不足、岗位越界、死亡编号和实际救命之间的拉扯。

### 铁道传习班第一批学员（群体）

| 项目 | 内容 |
|------|------|
| 来源 | 同人规划群体 |
| 身份 | 木匠、矿工、码头脚夫、识字归化民干部、工兵队混编学员 |

第一批铁道学员不培养工程师，而是培养能活下来的基层技术员。他们要先懂坡度、排水、超载和巡检，再谈蒸汽机和铁路管理。

这个群体的价值在于展示旅顺工业化的真实难度：不是缺一张图纸，而是缺一批愿意每天按表巡线、按规程停工、顶住上级催促的人。
