## 安楠

| 项目 | 内容 |
|------|------|
| 现职 | 北方医学专科筹备处负责人（自请出任） |
| 穿越前职业 | 公共卫生医师，在基层医疗机构和应急救援体系工作多年 |
| 穿越时状态 | 主动参与者 |
| 绰号 | 白萨满（黄骅的叫法，带调侃意味，安楠本人不置可否） |

### 穿越后经历

元老院早期，他进了卫生部系统，但始终和体制边缘保持距离。他不是那种愿意坐在行政会议上讨论规范文件的人，他更愿意去净化营、移民营、归化民聚居区，看病、教人、记录。

卫生部对他的态度是：能用，但管不住。他在部里有职衔，实际上大多数时间在外面跑。陈思根是少数真正懂他思路的人，两个人在卫生与营养的交叉地带合作了好几年。

旅顺议案出来之前，他已经在旧医登记和基层卫生员体系上做了相当多准备工作。004号通过以后，他主动请缨，没人比他更早把课程表写出来。

### 性格与行事方式

他不迷信任何一套医学体系，只问有没有用、死了几个。这让他同时得罪卫生部的正规医学派（说他不够严谨）和旧医圈子（说他把祖传医术当筛子过）。他对此无所谓。他认为北方第一年死于意识形态争论的人，和死于疾病的人一样冤。

对归化民学生和旧医，他的基本态度是：进账，能用则用，死亡必须有编号。不进账的，等于不存在，比没有更危险。

### 人物速写

- **外形**：偏瘦，手上有很多老茧，不是力气活留下的，是常年握手术器械和毛笔留下的
- **口头习惯**：在结束一段论述后常说"这个不会变"
- **弱点**：对行政程序缺乏耐心，有时候直接绕过而不是推动；在需要圆滑的场合容易说出太直接的话
- **关键一句话**："打旅顺只要赢一次。让人活下来，要天天赢。"

## 邓敏

| 项目 | 内容 |
|------|------|
| 性别 | 男 |
| 年龄 | D日（1628年）时约28-30岁；东北开拓（1637年）时约37-39岁 |
| 穿越前职业 | 地理教师（有野外勘探经历）→ 基层法律援助工作者 → 玩具厂合伙人（破产） |
| 穿越时状态 | 主动参与者，理想主义驱动加负债累累 |
| 现职 | 北方前线人员权益与牺牲审查小组 组长；兼任朝鲜矿产开发地质评估顾问 |
| 政治立场 | "文革左造"式原教旨共产主义者 |
| 标志性物品 | 铁盒（内有张志乾死亡证明及历年死亡档案）；褪色《毛主席语录》；旧军挎 |

### 相貌与体格

金丝边眼镜，极短寸头，面容酷似演员黄轩，但因长年野外勘探皮肤黝黑，显得更为坚毅粗糙。身高约一米七，体型精瘦结实，耐力惊人。手上满是老茧，手指却异常灵活，摆弄精密仪器和手搓应急工具都很娴熟。

### 出身与穿越前经历

军队大院子弟，成长环境培养了他朴素守纪的作风和对组织的深厚感情。师范大专地理学专业毕业，系统学习过地理、教育学、心理学，有扎实的野外地质勘探训练。

毕业后成为地理教师，违规带学生进行野外地质考察，遭遇山洪，造成重大教学事故，被学校开除。失业后没有立刻南下，而是在当地做了一段基层法律援助工作——帮工人、农民写材料、维权、建档案。这段经历没有给他带来法学学位，却给了他一整套文档、记录、追责的实操方法，以及一种深入骨髓的信念：死亡和伤害，如果不被记录，就等于不存在。

后南下广东与朋友合开玩具厂，遇全球性经济危机破产，背负巨额债务。走投无路之下，被"为了新世界"的口号吸引，加入穿越事业。

### 思想与政治立场

元老院中极少数的原教旨共产主义者。对马列毛理论有深入且系统的学习，坚信"左经日日读，功力日日深"。

他的档案室不是法律人的职业习惯，而是共产主义者的政治实践——从群众中来，必须记录谁因为谁的错误死去，才能"到群众中去"地追责。铁盒里张志乾的死亡证明，在他这里不是法庭证据，是革命债务。核心立场：

- **阶级立场**：旗帜鲜明站在无产阶级和被压迫者一边，死亡的追责首先是阶级政治，其次才是行政程序
- **民主集中制**：极度推崇党内民主和批评与自我批评，激烈反对元老院内日渐滋生的官僚主义和"新阶级"倾向
- **群众路线**：主动利用节假日在工厂、田间义务劳动，与归化民工人、农民建立了深厚友谊，在底层拥有极高个人威望
- **反压迫**：激烈反对任何形式的民族压迫、种族歧视和非人道的奴役制度

**隐患**：他已私下发展出地下基层组织与工会雏形——这是他长久被元老院主要权力体系排斥并处于严密监控下的核心原因。将他外派勘探队或安置在审查小组而非核心决策部门，是对他进行间接隔离的主要手段。

### 性格特点

在大多数场合，他不庆祝，不谴责，只记录。这比明确反对更让人不舒服——你不知道他站哪边，只知道他在记。

但这种沉默是策略，不是冷漠。一旦涉及死亡被掩盖、责任被抹平，他会立刻变成另一个人：脾气火爆，文风尖锐，喜欢用大字报和内部文章直戳问题核心。他对所有人的态度都一样：你可以做任何事，只要你接受这件事会被记录、被编号、在将来某个时候被追溯。

### 技能与特长

**地学**：地质勘探与找矿（擅长）、测绘与制图（精通）、水利与气象（熟练）、城市规划（半桶水）。来到旅顺方向后，承担朝鲜矿产开发线上的技术鉴定工作：赵大全从野外带回岩石样本，由他负责鉴定和品位初判，再交易冉武做经济建模。接受这个角色，部分原因是它要求精确和可核查——这和他做档案的逻辑完全一致。

**文档与追责**：档案管理、死亡编号体系、归化民证言保护、事故责任追溯，经由基层法律援助工作积累，以马列毛理论为框架，是他真正的核心技能。

**组织建设**：深谙群众组织、思想动员和基层组织构建方法，擅长凝聚人心。

### 在东北开拓行动中的角色

004号通过后，审查小组正式有了名分和独立报告权。这是他投赞成票的原因——不是支持鹿文渊，是支持档案室的合法性。他留守临高，主持北方前线死亡记录体系建设；朝鲜矿产地质评估，则由赵大全负责野外，他负责样本分析与初判。

**潜在戏剧冲突**：当主流路线因过于激进或脱离实际而导致危机时（奴隶暴动、生产停滞、士气崩溃），他积累多年的底层威望和务实的组织能力，可能让他从幕后走到前台收拾残局——也可能让他因"独走"建立基层组织而成为元老工作组内部激烈矛盾的导火索。

### 人物速写

- **气质**：保尔·柯察金式人物，铁盒和小红书同时在身上，前者对外，后者对内
- **口才**：在"只记录"状态下说话慢，每句话都经过筛选；在"原则问题"状态下文风辛辣，让人不舒服
- **弱点**：对"系统性改善"的耐心有限，有时候会在大局还在推进时提出让人窒息的细节追责要求；地下组织活动是他的阿喀琉斯之踵
- **关键语录**："否决很干净。可干净不能建立档案室。"

## 赵大全

| 项目 | 内容 |
|------|------|
| 现职 | 对外情报局北方站实地协调员（公开掩护借用黄骅贸易结构） |
| 身份 | 元老，穿越者；主动进入北方外线，后正式纳入对外情报局体系 |
| 出身 | 山东籍元老；穿越前经历待细化 |
| 现阶段任务 | 朝鲜方向情报与联络工作主责人；公开身份协助黄骅北方商路协调 |

### 经历与特点

他是元老，不是归化民。来到北方外线不是被谁提拔或收留，而是主动选择了最危险、也最容易建功的方向。沈阳这类地方没有临高的舒服日子，只有风沙、冻泥、查牌、盘问和随时可能断掉的暗线；他愿意在这里待着，动机里有很重的冒险欲和功业心。

他的优势不是专业知识，是人。他能在陌生的集市、渡口、村落里迅速找到话题、建立信任、打听出有用的东西，而且做完之后让对方完全不觉得被问过什么。

黄骅和他合作了好几年，对他的评价是：比大多数元老更会说人话。

鹿文渊最早是从黄骅口中得知赵大全在沈阳的工作价值。截至目前，鹿文渊并不是赵大全的直接上级，赵大全也不属于鹿文渊旧部系统；两人的关系应写成战略上互相知道、可能互相利用，而不是上下级或恩义关系。

随先遣队到达旅顺后，工作重心从辽东旧医摸底转向朝鲜方向。他是义州联络网的实际管理人，负责联络员的培养、任务下达和接触行动设计。他的情报判断不依赖书面分析，依赖的是对人的直觉——哪个联络员真可靠、哪条消息是真的、哪次接触背后有人在试探，他往往比正式情报分析早半步感觉到。

### 与旗铜委的工作关系

对外情报局（赵大全）与政保局（旗铜委）在朝鲜业务上形成制约关系：赵大全决定怎么接触、接触谁；旗铜委审查接触对象的政治可靠性，有权否决某个联络员的使用资格。两者互不统属，摩擦是结构性的。

### 人物速写

- **外形**：壮实，山东人的骨架，晒得很深，在人群里看着像普通力工或小商贩
- **智识**：不读书，但记性极好，尤其是人名、面孔、关系链条。见过一次的人，三年后能认出来
- **弱点**：在正式场合会不适应，让他写报告就漏掉一半——他能说出一百件有价值的事，落到纸上只剩三分之一。旗铜委对他"凭感觉"的部分持保留意见。
- **关键关系**：与黄骅长期协作、互信很深，但不写成隶属或感恩；对安楠保持距离但尊重；与鹿文渊目前不是上下级关系，鹿文渊更多是通过黄骅认识到他的价值

## 林剑行

| 项目 | 内容 |
|------|------|
| 元老编号 | C-0778 |
| 现职 | 广东大区高等法院法官 兼 大区法务办公室高级顾问 |
| 穿越前职业 | 某211大学法学专业，持法律职业资格证（A证） |
| 穿越时状态 | 奠基元老，携家庭成员集体入组 |
| 家庭 | 已婚，育有一子；直系亲属为法学会核心成员，司法理念上与他既合作亦有分歧 |
| 政治派别 | 扩张派激进翼；皇汉社核心成员（内部"文化塑造派"与"制度整合派"代表声音） |

### 核心政治立场

**文明主体论与大陆干预主义**：坚信澳宋文明内核必须是汉文明，对"等后金入关再决战"的战略定力派深恶痛绝，称其为"达利特领班"。他认为这是对历史的不负责任——必须在1644年前主动粉碎内亚军事集团问鼎中原的可能性。

**"新中华"民族构建**：主张最高权力绝对掌握在汉族元老和高度汉化的归化民精英手中；对治下非汉族群推行半强制性的语言、文字、风俗、国家认同的全面整合。但在实践层面坚守一条原则——"不认同抽象的民族，但善待具体的人"。

**自我定位**：自称"新保守主义者"，简称"劳保"。"保"的是辛亥革命精神，以及德式施特拉塞派（注意：明确区别于希特勒派）强调的国家至上、劳资合作理念。他的经典吐槽是："在十七世纪，纳粹来了都得发现自己变进步派了。"

**历史观**：持"1644史观"，视元、清为征服政权而非正统中华王朝。另有一套"日耳蛮学"理论框架，认为印欧文明的种姓结构是其根本缺陷，并以此批判元老院内崇拜北约或旧日本帝国模式的人。

**体系设想**：反对单一制大一统帝国，主张以澳宋为核心建立"新中华诸夏体系"——通过分封、附属国、紧密经济军事联系，形成类似五眼联盟的文明圈，比大一统更有活力和扩张性。

### 工作实践

在广东法院任职期间，以"好说话"著称——对因公越界的外派站人员持宽容态度，认为开拓时期一线灵活性至关重要。有过动用元老马车紧急送受伤黎族孩童就医并垫付费用的记录，多次亲自带队深入黎区做普法调解。他的逻辑始终如一：给归化民汉化的路径是元老院的责任，但这不等于权力分享的起点。

他同时是临高BBS上的活跃笔战者，和等待派、cosplay北约派长期论战。

### 参与旅顺行动的隐秘动机

他是东北人——另一个时空的东北人。这件事他几乎从不对外人说。

他记得那片土地冬天的雪、秋天的海风，记得城市和山峦的轮廓。在他的1644史观里，后金（清）是导致那片土地沉沦的源头。支持独走北伐，不只是意识形态驱动，也是因为这是他唯一能为那片失落的故土做的事情。历史复仇与文明救赎，在他这里是同一件事。

### 与旗铜委的潜在张力

旗铜委祖上是旗人，虽已改汉入籍，但这个背景在林剑行的框架里不是可以完全忽略的东西。林剑行不会在明面上对旗铜委有任何不友好的表现——原则是"善待具体的人"，旗铜委的工作也无可挑剔。但如果将来某个时刻，涉及满族相关人口的具体处置方式出现分歧，两个人站的位置大概不会相同。这条潜在裂缝，目前还在水面以下。

### 人物速写

- **外形**：职业法律人气质，整洁，说话有条理，在一群前线气质的人里面显得偏"文官"
- **矛盾之处**：怀有重塑文明的崇高使命感，同时坦然享受元老待遇，会自嘲"离不开南海农庄的牛排和24小时热水澡"
- **弱点**：意识形态框架有时导致他在具体问题上过早下结论；对等待派的公开鄙视在某些场合会让自己在政治斡旋中吃亏
- **关键语录**："不认同抽象的民族，但善待具体的人。"

## 易冉武

> **名字说明**：故事线文档中部分写作使用"易冉吾"，与此处"易冉武"为同一人物，系笔误/版本不一致。统一以"易冉武"为准。

| 项目 | 内容 |
|------|------|
| 现职 | 商务部/统计局系统（具体职衔待定），随先遣队赴旅顺 |
| 穿越前学历 | 某985高校控制工程专业硕士；本科室友学经济，常被拉去蹭课 |
| 穿越时状态 | 主动参与者，带着"看乐子"的心态加入，穿越后既来之则安之 |
| 政治派别 | 皇汉社核心成员 |
| 政治倾向 | 威权主义、实用主义 |

### 穿越后经历

他不是带着使命感来的，是带着好奇心来的——穿越前觉得有意思，穿越后发现控制工程在17世纪用处有限，在基本劳动力阶段待了一段时间，然后想起本科室友拉他蹭过的那些经济课，转头往经济口发展。先后在统计局和商务部任职，做过数据整理、贸易核算、物资估算一类的工作，属于那种不声不响把事情做准确的人。

他有社恐，融入集体慢，但一旦融入就是真朋友，对朋友热心，不会敷衍。在五道口内人缘不错，不是因为他善于社交，而是因为他真的记事、靠得住。不轻易发表对局势或人物的看法，喜欢静观其变，等别人说完再自己想。

### 核心意识形态

他是皇汉社成员，但不是林剑行那种爱打笔仗的理论型。他的立场是威权主义加实用主义——不关心旗帜和口号，关心机制能不能跑起来。

在民族政策上，他主张以汉文化、汉民族为绝对主体推进国族构建，趁民族主义尚未兴起就把其他民族的民族意识消弭掉，整合东亚、北亚、东南亚诸民族进入统一框架。激进社会改造、消灭封建地主经济是他认为必须做的配套工程，不做这一块，文化整合就是空的。

在北方战略上，他主张早期干预，打断满清历史进程，阻止其入关，在辽宁和朝鲜建立军工基地以同时钳制满清和大明，对"等待后金入关再决战"的主张持激烈批评态度。

### 与旗铜委的关系

### 穿越后经历

范云淮不是元老院里最耀眼的技术专家。建设工程管理大专出身，在旧时空学过的东西不算尖端，但覆盖面很杂：农田水利、市政项目、施工组织、特种设备安全、电工基础，哪一项单拿出来都压不过专业口的大拿，合在一起却正好能填旅顺最缺的空。

穿越后，他一度在几个部门之间来回挂职。国土交通省要他看道路、住房、供暖和基础设施；农林水产省要他懂一点田间水利；经济产业省又看中他接触过特种设备安全管理。于是他成了一个很少见的"三不靠"人物：哪个口都能说上几句，哪个口都不完全把他当自己人。

004号方案通过后，他判断北方开发会长期吃紧，主动要求加入下一批北上支援队伍，常驻旅顺。决定以后，他开始把D-Day以后很长一段时间里工业口、农业口、建筑口和其他部门欠下的人情债一笔笔收回来。过去谁让他帮忙跑过工地、补过手续、带过短训、救过临时项目，现在就得在北方项目上还一点。十八名归化民干部、一辆旧时空自行车、一套DSZ05水准仪、两把9mm左轮和三十六发子弹，就是他这段时间的成果。

有人说他太会做人，所以原则模糊；又太不会做人，所以总能把原本能圆过去的组织摩擦弄成明账。范云淮自己不太承认。他觉得自己只是在给北上派争话语权，也是在给旅顺第一年多凑几把能用的锹、几双能指挥人的手。

### 职责定位

他的项目名叫"东北亚经略生产生活保障性工程"，听起来像一个什么都能往里装的筐，实际上是旅顺落地以后最不体面的硬活：

- 指导并参与现阶段住房、供暖建设，避免第一冬把人冻死在临时棚里
- 指导并参与农田水利建设，解决田块排水、灌溉、冻土和春耕准备问题
- 指导并参与生产生活用水建设，包括井、管沟、水池、排污和临时净水设施
- 参与基础设施补缺，哪里缺路、缺桥、缺排水、缺工棚，就先把哪里补到能用
- 处理建设过程中随时冒出来的工程、材料、劳力和安全问题
- 参与铁山工业区特种设备安全管理体系建设，不管高炉怎么炼铁，但要管锅炉、吊装、压力容器、用电和事故登记
- 组织编写因地制宜的短期培训教材，联合当地开短训班，补"人才荒"和"干部荒"

组织上，004号执行筹备组可以暂设**工程建设局（筹）/交通工程合署办公室**。这个牌子之所以能合署，是因为交通总局和临高建筑总公司同属国土交通省系统，北方前期又没有条件把道路、工棚、码头接口、轻便轨施工协调和铁道标准完全拆开。范云淮常驻，管现场施工、生活保障、材料人手和各口接口；柳晴川非常驻，出差或轮驻时在同一块牌子下办公，管铁路线路标准、验收、教材和铁道科课程。两人不互相隶属，铁路技术标准仍由柳晴川签，现场资源和施工协调由范云淮处理。

他不是铁山工业区的总工程师，也不是农业局局长。他的价值在于把分散在几个系统里的低层工程问题串起来，让前线不至于每遇到一条沟、一口井、一台锅炉、一间冻裂的宿舍都临时找人。

### 性格与行事方式

范云淮有一腔为元老院建功立业的热血，也有很现实的自知之明。他知道自己不是顶级专家，所以特别重视"能找到谁"和"能带来什么"。他讲话常常先报资源，再谈理想：几个干部、几套工具、多少天能教会、出了事谁签字。

问题也在这里。他太习惯用旧账、人情、借调、挂职和临时协调解决问题，容易让正式条线觉得他在绕程序；但真到了需要圆滑收尾的时候，他又会把账摊得太明，说这个干部是谁欠他的，那套设备是哪次帮忙换来的，哪件事原本该哪个部门担责。于是反对者说他"既太会做人，又太不会做人"，听起来矛盾，放在他身上却很准确。

他对东北实际情况了解有限。除了知道山东有鹿文渊元老、济州岛有根据地，很多情况都是两眼一抹黑。但他相信在战无不胜的文德嗣思想引领下，总能把障碍一件件拆开。这个信念有时候显得天真，有时候又是他能在一堆坏消息里继续往前推的燃料。

### 在东北开拓行动中的角色

范云淮适合放在第二批或后续增援队伍里登场。他不是第一个抢滩的人，而是抢滩后发现"能住、能烧水、能排水、能开工、能不死人"全都还没解决时，被各方想起来的人。

他的到来可以带出几条具体戏：

- **住房供暖线**：归化民宿舍、元老临时驻地、工棚和冬季供暖之间的优先级冲突。
- **农田水利线**：春耕窗口很短，排水、灌溉、冻土、牲口和劳力调度全都要抢时间。
- **用水卫生线**：与安楠在饮水、排污、净化营卫生和传染病预防上形成交叉。
- **铁山安全线**：工业口追产能，范云淮追安全规程和事故登记，两边都觉得对方耽误大局。
- **短训班线**：十八名归化民干部不是够用，而是一个火种。怎么把他们拆成班组长、测量员、施工记录员、水利员、电工学徒，是他最实际的工作。

他给北上派带来的不是一票决定性力量，而是一种很有用的存在感：旅顺每多解决一个生活保障问题，北上派就多一条"不是只会喊战略，也能把人安顿下来"的证据。

## 柳晴川

| 项目 | 内容 |
|------|------|
| 现职 | 国土交通省/交通口铁道技术顾问；东北亚铁路建设与铁道教育专项顾问 |
| 技术专长 | 铁路线路、路基、桥涵、轨道标准、工矿窄轨、调度与维修制度 |
| 驻扎状态 | 不能长期常驻东北；以出差、轮驻、专项指导方式参与旅顺铁路线 |
| 工作职能 | 建设指导、技术标准制定、归化民铁道人才培养、铁道课程体系设计 |
| 教育参照 | 以"铁路建设带动工程教育"为逻辑，参照哈工大建校渊源设计旅顺铁道教育线 |
| 角色定位 | 非常驻元老顾问；铁路专项的技术和教育牵头人，不占东北局日常行政主岗 |

### 穿越后经历

柳晴川是交通口里少数真正懂铁路的人。这个"懂"不是会背几条铁路史，而是知道铁路为什么不是两根铁条铺在地上就能跑：路基排水、坡度、曲线半径、桥涵、轨枕、道床、检修、调度、装卸站和人员制度，少一个环节，前面修得越快，后面返工越疼。

在临高体系里，铁路长期不是最先能吃上资源的项目。海运便宜，驿道能凑合，短距离搬运可以用手推轨、牛车、板车和人力。柳晴川因此很长时间都处在一种"知道将来很重要，但眼下轮不到"的位置。他不急着争口号，反而更愿意做标准和教材：轨距怎么定，轻便轨能做到什么程度，什么样的工人能当线路工，什么样的木匠能转轨枕工，调度员最少要认哪些表。

旅顺口方案让他的铁路知识第一次有了现实落点。铁山、旅顺城区、西港、净化营、仓储区和金州方向之间，全靠车马和人工，会把煤、铁、粮、木料、病人、伤员和炮架全部堵在路上。柳晴川判断，旅顺第一阶段不需要幻想标准轨干线，但必须尽快建立可维护、可扩展、可教学的工矿窄轨体系。

问题是他本人不能长期在东北驻扎。临高和南方交通口也离不开他，铁路相关人才太少，让他常驻旅顺等于把其他方向的技术判断抽空。因此他的参与方式被定为出差：关键节点北上，做踏勘、定标准、开课、验收，然后离开；日常执行由旅顺本地干部和归化民技术员接住。

### 铁路职责定位

柳晴川的职责不是"亲自把铁路修完"，而是让旅顺有能力持续修、持续用、持续培养人。

- **线路踏勘与方案裁剪**：优先处理铁山—西港—仓储区—城区之间的工矿窄轨，论证金州方向是否先做军用轻便线。
- **轨道标准与维修制度**：确定轨距、轨型、枕木、道床、坡度、弯道、排水、桥涵和日常巡检制度，避免每段线都按临时工地办法乱铺。
- **装卸与调度体系**：把车站、货场、煤栈、铁料堆场、粮仓和净化营转运点纳入统一调度，而不是单纯修一条"能跑车"的线。
- **设备转运与复装验收**：审定机车、道岔、检修设备从临高出厂到旅顺复装的拆装清单、吊装方案、备件包和试车标准，强调不能把接近30吨的轻型机车当普通货物发运。
- **铁道人才培养**：在旅顺学府或单独传习所下设铁道科，培养线路工、测量员、桥涵工、轨道维修工、转运站调度员和工矿车辆修理工。
- **标准轨预研**：不在第一阶段强推标准轨，但保留测绘、路权、港区接口和未来干线扩展条件。

他的原则很硬：临时线可以临时，但临时标准不能胡来。否则一旦铁山产能起来，铁路就会从解决问题的工具，变成制造事故的机器。

### 与范云淮的边界

范云淮管生产生活保障：住房、供暖、饮水、排污、农田水利、临建、短训班和特种设备安全。柳晴川管长期运力系统：线路、轨道、桥涵、调度、维修和铁道教育。

两个人会频繁合作，也一定会吵。范云淮会倾向于先铺一段轻便轨，把煤和粮拉起来，"将就能用就得了，反正后面要重建"；柳晴川会追问这段路基排水怎么做，坡度能不能让满载车安全下行，枕木三个月后烂了谁换，调度员看不懂表怎么办。

这种冲突不是私人恩怨，而是旅顺工程系统的两种真实压力：一个要救眼前，一个要防未来。

### 铁道教育线

柳晴川最看重的不是第一条铁路，而是第一批能维护铁路的人。

他提出的铁道教育线可以参照哈工大建校渊源的逻辑：铁路建设不是单个工程，而是会逼出一套工程教育体系。旅顺可以从"铁道传习所"起步，先教识字、算术、测量、制图、材料、线路养护和调度表，再逐步并入旅顺学府工程技术学院，成为"铁道科"。

第一批学生不需要成为工程师。他们要先成为可靠的线路工和技术员：知道雨后先看哪里，知道桥涵堵了会怎样，知道煤车不能随便超载，知道一张错写的调度表能害死几个人。等这批人长起来，旅顺才有资格谈更长的铁路、更重的车辆和更远的干线。

### 人物速写

- **外形**：交通口技术干部气质，随身带图纸筒和厚笔记本；看路时比看人认真
- **说话方式**：喜欢把宏大战略拆成坡度、排水、桥涵和检修周期，听起来扫兴，但通常是对的
- **长处**：铁路系统观强，知道工程、调度、维修和教育必须一起设计；不迷信一步到位
- **弱点**：不能长期驻扎，现场威信需要靠阶段性成果维持；对临时凑合工程容忍度低，容易被前线认为不接地气
- **与范云淮的关系**：互相需要，互相嫌弃；一个解决今天怎么活，一个追问三个月后会不会塌
- **关键语录**："铁路不是铺出来的，是养出来的。没人会养，铺得越长，欠账越长。"

## 旗铜委

| 项目 | 内容 |

| 民族登记 | 汉族（原为满族，祖上改汉入籍） |
| 穿越前出生地 | 东北某城 |
| 穿越时年龄 | 约24岁 |
| 穿越时间 | 2008年夏 |
| 穿越方式 | 非自愿，暑期跟团海南穷游，误上文总船 |
| 学历 | 理学学士，应用数学 |
| 现职 | 政保总局北方安全监察组派驻代表 |

### 穿越经历

他不是主动去的。

2008年夏，工作两年，攒了点钱，跟团去海南穷游。团费便宜，船票便宜，同船的人形形色色。其中有个姓文的，话不多，看着像搞研究的中年人。等船开出去，海南已经消失在视线里，那个姓文的才笑了笑，说了句让他终生难忘的话——大意是，欢迎加入。

他以为是什么传销。

等搞清楚自己在哪里，已经是大明崇祯元年了。

### 穿越后：震惊、崩溃、沉沦

他不是理工科的，不会开机床，不会修内燃机，不会给人看病。应用数学本科，在17世纪能干什么？建模？建什么模？给谁建？

元老院早期的那批核心事件他全没赶上——不是没机会，是没心情。在芳草地的宿舍楼里坐了很久，后来在文理学院找了个数学教职，一教就是好几年。数学课不愁学生，尤其女学生。

他不算彻底堕落，只是选择了一种没有目标的生活。女仆、女学生、黄酒、围棋，日子可以过得很滑，滑到不用想明天。偶尔有人找他谈未来，他都笑笑敷衍过去。

因为不是主动穿越的，早期穿越众的一系列大事件均未参与。元老院登记在册，实质上是个体面的酱油。他自己也这么认为，而且认得很坦然。

### 身世与血脉

他家是东北人，但根子更复杂。

父系来自关外，祖上是旗人。清朝倒了之后改了汉名，入了汉籍，到他这辈已经两三代"汉人"了。但家里还留着一些东西：一个模糊的旗姓，几句没人听得懂的零碎满语，和一种说不清楚的、对"关外"的隐约认同。

母系来自山东，外婆辈闯关东落在辽宁。

所以他这个人，身上同时有山东大汉的骨架和东北蛮子的血气，站在那里一米八几，肩膀宽，声音大，脸皮厚。开口聊天常常让人不知所措——情商确实很低——可大家慢慢发现，他说的很多东西事后都应验了。那是学数学的人特有的一种扫兴：他不说漂亮的话，他说准确的话。

### 内心的小小波澜

东北攻略开始的时候，他比任何人都沉默。

不是因为不关心。是因为太关心了，反而不知道说什么。

那片地方在他上辈子的记忆里是高速公路、麦当劳、冬天冻手的铁门把手、春节时炕上的土豆炖排骨。在这个时代，那里是辽南、后金、皮货、马市、死人堆里长出来的边疆。他某个祖先在那片土地上吃过什么苦、挨过什么打、活了多少年——他不知道，也没有人告诉他。

他只知道他想去看看。

这个念头让他不舒服。一旦去了，就不能再当酱油了。当酱油的人随时可以说"我不行的，我就是个教书匠，你们厉害你们去"。但去了东北，就是要做事情，要让人放心把某些权力交给你，要对某些人的死活负责。

他在芳草地磨蹭了很长时间。

最后，他跟那几个跟了他好几年的女仆说，我去一趟，不一定回来。

女仆们给他收拾了行李，送到码头，顺便塞了一坛他平时喝的黄酒。他拎着坛子上了船，站在船尾，看着芳草地的灯光一点一点消失在海平线以下。

### 政保局的人

他去东北，不是以元老身份去旅游的。

政保总局看中了他，理由不复杂：识字，懂数学，对东北有天然情感联结，元老院里没有明显派系归属。一张白纸，而且已经被晒了十多年，不会轻易染色。

祁局长亲自谈的。

他后来想，祁局长大概是算过的。一个数学背景、中年才开窍想进步的人，是最好用的人——不会因为感情用事做蠢事，也不会因为野心太大做坏事。进步，但是稳健的进步；可用，但是可预期的可用。

他进了政保局，开始学这辈子从未认真学过的东西：档案管理、监察程序、线人筛选、密电码本，以及如何在一张无聊的晚饭邀请里听出谁在跟谁递消息。

他发现自己干这个有点天赋。

应用数学的底子，说到底是在复杂数据里找模式。人际关系也是一种数据，只是变量更脏、噪声更多，结论更难看。

### 在朝鲜矿产故事线中的角色

旗铜委是**政保局**派驻，职责是内部政治安全，不是对外情报的主责人。在朝鲜矿产开发业务中，他的介入点是：

- **审查赵大全拟使用的朝鲜联络员**：核查联络员有无与金自点派系或清朝使节的接触记录，有权否决特定联络员的使用资格
- **监控旅顺内部的信息暴露面**：参与朝鲜业务的旅顺人员中，有没有渠道外泄？
- **与赵大全的结构性张力**：他管内部安全，赵大全管外部接触，边界天然模糊。他压缩赵大全的行动空间，赵大全认为他过于谨慎妨碍业务。这个摩擦不是个人恩怨，是体制设计的产物。

### 人物速写

- **外形**：高大，一米八几，肩宽背厚，在关外晒了几个月之后皮肤比芳草地时更深。站在人群里看着像个搬货的，开口之后让人想起某种精密仪器。
- **口才**：差。但沉默的时候让人觉得他在想什么，说话的时候让人觉得他不太会说话。两种状态都让人轻微不安。
- **智识**：数学底子，逻辑清楚，善于在混乱信息里找结构，尤其是数字类和行为模式类分析。
- **政治立场**：向左，向内，想进步，但不激进，也不表演。更接近祁局长式的务实左翼，而非意识形态表演者。十多年酱油生活，人到中年，思想逐渐左转，目标是"想进步"，具体路径参照祁局长。
- **弱点**：情商低，社交能力差，习惯了酱油生活，遇到需要圆滑处理的人际场合容易短路。他清楚自己的弱点，但改不了，只能靠沉默来掩护。
- **关键关系**：
  - 对旧时期的女仆们有真实感情，这是酱油岁月里少数真实的东西
  - 对祁局长有真实的敬畏，不是表演
  - 对鹿文渊保持职业距离——他是政保的人，职责决定立场
  - 对旗阿其有一种说不清楚的神秘联结，目前未公开
  - 和易冉武在数据方法论上有共鸣（一做安全情报，一做经济统计）
  - 和林剑行之间存在旗人背景引发的潜在张力（已在林剑行条目中说明）
  - 和安楠在医疗死亡记录上有交集
- **在北方人事格局中的位置**：政保总局派驻监察代表，负责北方安全档案、涉密人员审查、事故责任追溯记录。不是最显眼的人，但他签字的地方，别人不能绕过去。

## 目前无背景或背景真空的重要岗位

### ① 招远金矿驻局监察员
草稿里明确存在（第50章第一摞六个"较容易"坑位之一），但完全没有点名人选和派系。金矿是中央银行贵金属储备的命脉，是整个004号债券的抵押核心，谁占这个位置极其敏感。

### ② 移民安置与净化营管理主任
第52章明朗选了一个有黄骅商路履历的人，但明朗本人的评估用词是"条件符合"，没有任何实质派系背书——叶雨茗系不满意、天地会没渗进去、鹿文渊也没有明确收编。这个岗位管移民生死，是东北局最容易出大事的地方。

### ③ 管制区行政协调次席
第52章里明朗直接写了"条件待明确"，明确表示现在写任何名字都太烫手。这是东北局未来行政体系的二把手预备位，目前是硬性真空。

### ④ 铁山工业区联合筹备处实际负责人
展无涯的工业口、林深河的军工口、化工口三方都在争，但"联合"意味着谁都没有完全到手——驻场的实际执行负责人目前没有具名。

范云淮可以填补铁山的**特种设备安全管理与生产生活保障接口**，但不宜直接写成铁山工业区联合筹备处的实际负责人。他能管锅炉、吊装、用电、事故登记、工棚、道路和用水，不能替代工业口或军工口对冶金、军工工艺和产能路线的主责。

### ⑤ 教育与文化整合系统
林剑行有《归化民法律地位初步意见》，但这是法律框架，不是教育行政。东北局教化归化民、整合辽东原住民这条线目前没有任何专门负责人，是被安楠的医学体系和林剑行的法务线覆盖但都不主导的空白地带。

### ⑥ 铁路与铁道教育线
铁路不适合被压进普通工程建设线。范云淮可以处理短期轻便轨施工协同和基础设施接口，过渡期也可以与柳晴川在工程建设局（筹）/交通工程合署办公室同牌办公；但铁路的线路标准、路基桥涵、调度维修和铁道教育需要柳晴川作为专项顾问。柳晴川不能长期驻扎，因此这条线天然带有"顾问出差—本地接班—制度能否留下"的张力。

## 备注
最关键的真空曾是**①招远金矿监察员**（财政命脉）和**③管制区行政协调次席**（未来行政接班人预备位）。目前易冉武、林剑行已经分别切入这两个位置。范云淮新增后，**工程建设与生产生活保障线**有了可用抓手，但铁山工业主责和农业生产主责仍不应被他完全吃掉。柳晴川新增后，铁路可以独立成线，但因为他非常驻，铁道教育和本地干部接班必须成为铁路故事线的核心问题。
