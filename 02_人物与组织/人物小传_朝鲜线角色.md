# 旅顺口同人 · 朝鲜线角色人物小传

> 本文件收录旅顺口线中朝鲜宫廷、沈阳馆、义州商路及朝鲜地方社会相关的重要外部角色。  
> 这些人物不是元老，也不是澳宋归化民；他们的核心价值在于让“朝鲜被澳宋搅动”落到具体的人、文书、风险和选择上。

---

## 尹时晦（暂名）

| 项目 | 内容 |
|------|------|
| 来源 | 同人新增 |
| 阵营 | 朝鲜王朝 |
| 身份 | 承文院判校，兼备边司文书差遣；可按正文需要微调为承文院参校或承政院副承旨路线 |
| 官阶定位 | 不入真正决策核心，但不是低级吏员；能接触事大文书、清方来往、沈阳馆奏报和世子相关案卷 |
| 核心定位 | 被澳宋制度震动的朝鲜体制边缘高阶官僚；赵大全与对外情报局发展的宫廷内部代理人 |
| 关系入口 | 赵大全、方才、江山、韩允直、韩允成、李汪、朝鲜备边司、承文院、沈阳馆 |
| 旅顺功能 | 朝鲜宫廷代理人伏笔、李汪线验证、汉城文书线升级、澳宋影响朝鲜政治的长期钩子 |

尹时晦出身两班旁支，读书、应举、入仕都走正路，却一直停在“能写国家大话，不能决定国家命运”的位置上。他不是失意狂生，也不是低贱小吏。他熟悉朝鲜朝廷的每一道文书流向：给清国的咨文怎样写得恭顺，给沈阳馆的私信怎样写得含糊，备边司诸臣怎样把责任塞进看似稳妥的句子里。他的才能足够让上官放心用他，门第和性格又不足以让他真正进入权力中心。

丙子之役以后，他对朝鲜朝廷的看法发生了变化。朝廷仍在说礼义、名分、宗社，可他每天誊写和修订的都是屈辱、推诿和互相保身。他不是不忠，也不是亲清；恰恰相反，他太清楚自己忠于的这个国家已经学会了用漂亮文辞掩盖无力。

澳宋最初进入他的视野，不是通过奇货，也不是通过金银，而是通过济州、赎俘、商账和沈阳馆之间流动的文书碎片。他看见澳宋会登记人口、会清点粮食、会给官奴婢改身份、会把伤病、死亡、工役和配给写成能追溯的账。对尹时晦来说，这比水银镜和澳洲货更可怕：海寇可以剿，商人可以查，奇技淫巧可以禁；但一种能把人重新编入制度的力量，不能只靠一纸禁令挡住。

他被澳宋影响，不是因为相信澳宋仁慈，而是因为他发现澳宋有效。朝鲜朝廷把世子、臣子、商人和贱民都写进名分里，澳宋却把他们写进表格、船单、粮册、诊籍和死亡编号里。尹时晦最初鄙薄这种粗陋，后来开始恐惧，最后不得不承认：粗陋的东西有时比优美的东西更能活人。

### 与赵大全和对外情报局的关系

赵大全不应直接“收买”尹时晦。更合理的路径是三步：

第一步，韩允直用掉一次机会，只传回“汉城可受文者在承文院，不在沈阳馆”的方向。赵大全据此判断，朝鲜宫廷内部已有可接文书的人，但不能贸然压上任务。

第二步，方才接手黄骅旧网后，把尹时晦列入对外情报局档案。档案名目不能写“内应”，只能写“可发展对象”。赵大全重视他的手感，方才重视他的结构位置，两人都看见这个人有价值，但判断角度不同。

第三步，尹时晦第一次发挥作用，不是传出大秘密，而是在一份涉及“义州外柜”和“沈阳馆商贾往来”的文书上改动案名。他不抢韩允直已经改软的措辞，而是把“另案查核”压回“沈阳馆日用项”，让追责无法立刻升级成独立案卷。表面看只是文书分类，实际效果是让韩允成、韩允直和李洛由外柜这几条线暂时不被一网打尽。

赵大全由此确认：这个人不是澳宋的人，但他已经开始按澳宋给他的外部压力重新计算朝鲜内部的文书后果。

### 与李汪线的关系

尹时晦不能直接替李汪做主，也不能变成“拥立世子”的热血官员。他的价值在于判断：李汪身边到底有没有人，汉城是否有人愿意在不明说的情况下替世子留一条文书退路。

如果李汪主动递出第一句话，尹时晦可以成为验证这句话的人。他不必见李汪，也不必忠于李汪。他只需要在汉城文书里看见：哪一派想杀他，哪一派想用他，哪一派只想把他继续放在沈阳当人质。尹时晦看见的不是王位，而是王位周围的责任如何被写出来、擦掉、转嫁。

### 性格与弱点

尹时晦外表克制，文辞干净，说话不急。他很少直接表态，习惯先问“此语入何案”“此案归何司”“日后谁可追问”。这种人天然适合做文书官，也天然适合被赵大全看中。

他的弱点是过度相信文字能改变风险。他知道一句话可以救人，也知道一句话可以杀人，于是容易高估自己在文书上的腾挪空间。赵大全会提醒他：文书能拖住刀，不能永远挡住刀。方才则会更冷，把尹时晦视为一个高价值杠杆，而不是一个需要拯救的同情对象。

他对澳宋的感情也必须保持复杂。他不崇拜澳宋，更不愿承认自己已经被影响。他会批评澳宋粗鄙、无礼、重利、轻名分；但当朝鲜朝廷再次用漂亮文字掩盖牺牲时，他会下意识想起澳宋的表格和死亡编号。

### 后续戏份

1. **第一次回声**：58章以后，韩允直线传回“可受文者在承文院”的暗示，赵大全和方才分别做出判断。尹时晦不必正面出场，只以档案名出现。

2. **第一次出手**：汉城内部讨论“义州外柜”“沈阳馆商贾往来”时，尹时晦把“另案查核”压回“沈阳馆日用项”，让追责无法直接落到韩允成和韩允直身上。读者第一次看到澳宋确实搅动了朝鲜宫廷内部。

3. **第一次主动发问**：他通过韩允直或另一条新壳，递出一个不是情报、而是政治问题的问题：若朝鲜终有一日不可保，澳宋保的是王，还是人？这句话是朝鲜线真正的长期钩子。

4. **旅顺后的连接**：旅顺口建立后，尹时晦的价值上升。他不再只是沈阳馆外的文书眼，而是能帮助澳宋判断朝鲜宫廷对旅顺、济州、义州、世子和清国压力的真实反应。

### 写作注意

- 不要把尹时晦写成“被澳宋先进性折服”的单向投靠者。他是朝鲜官僚，被澳宋震动，也被澳宋威胁。
- 不要让他太快与李汪合流。李汪线的关键仍是“李汪自己先递第一句话”，尹时晦只负责验证汉城是否有承接这句话的文书空间。
- 不要把他降格成普通线人。他的戏份重点不是偷听密谈，而是改写、缓写、压案、转责，让朝鲜政治在纸面上发生微小但致命的偏移。
- 不要让赵大全亲自支配他。赵大全能判断人，方才能建档案，对外情报局能设计接触，但尹时晦必须保留自己的恐惧、傲慢和选择。

### 人物速写

- **外形**：四十岁上下，身形清瘦，常年伏案，手指有墨痕；衣冠整洁但不华贵。
- **口头习惯**：不说“我以为”，常说“此语入案之后”。
- **关键能力**：熟悉朝鲜宫廷文书流向，能判断一句话进入哪份案卷后会伤到谁、保住谁。
- **关键矛盾**：忠于朝鲜，却越来越相信朝鲜自己的文书救不了朝鲜人。
- **关键一句话**：“海寇夺城，尚有城可守；若一国之人先入了别人的册，城在不在，又有何别？”
