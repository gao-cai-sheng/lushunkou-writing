# 进击旅顺口（旅顺要塞城市群）游戏设定文档

## 《临高启明》衍生游戏 · 要塞建造与经营模拟


> **时代背景**：公元1640年（明崇祯十三年，清崇德五年）。小冰河期已入最寒一段，皇太极在沈阳称帝四年，松锦决战的阴云在辽西铺开。元老院于天聪七年（1633）后金占领旅顺之后的第七年，借伏波军北上之势，从济州岛、屺坶岛、镇江堡三路并进，于崇祯十一年（1638）的"辽南攻略"中夺回旅顺口，并以**金州地峡棱堡线**为陆上屏障，将整个辽东半岛南端约2500平方公里土地纳入实际控制。
>
> 旅顺口由元老院辽南军政委员会直辖，是元老院在东北亚最大的单体殖民据点、海军前进基地、工业飞地与远东情报中枢——也是从屺坶岛"桥头堡"升格而来的真正的"北方根据地"。规划归化民人口10万、正规军1.2万、元老常驻45人。



## 一、辖区总体布局

### 地理概况

**辖区规模**：北界**金州地峡**（南关岭—大黑山一线），南至**老铁山**，西临渤海湾（双岛湾、羊头湾），东抵黄海大连湾。南北纵深约65公里，东西最宽约35公里，最窄处（金州地峡）仅约3公里。总面积约2500平方公里。

**地势**：辽东半岛南端为典型的低山丘陵海岸地形。西部为**老铁山山脉**主体，最高峰海拔约465米，山势陡峭，南端直插渤海，是渤海海峡的天然灯塔位置。中部为低缓丘陵与河谷盆地，土壤较薄但适宜耐寒作物。东部沿黄海岸线多礁石与小型海湾。

**核心港湾**：

- **旅顺港**（东港、西港双港体系）——天然不冻深水良港，港口仅以"老虎尾"半岛与黄金山之间约300米宽的水道与外海相通，内港水域开阔约5平方公里，是元老院北方海军总锚地。
- **双岛湾**——西北侧次要军港，作为伏波军补给与备用泊地。
- **大连湾**——东北侧浅水大湾，目前仅作渔业与小型船只活动区。

**气候特征**：比屺坶岛更严寒。年均气温约8℃，1月均温-6℃，极端低温可达-22℃。冬季冻土深度约1米，海港**主航道全年不冻**（地理学奇迹之一），但内港浅水区每年1—2月有短期结冰。建筑必须做防冻处理，地基深度不少于1.5米。

**总体分区（地理顺序，由北至南）**：

1. **金州地峡防线区**——陆上唯一门户、棱堡群
2. **金州城与北部农垦区**——粮食与煤炭主产地
3. **铁山工业区**——重工业、军工核心
4. **旅顺城区**——行政、商业、归化民中心居住区
5. **旅顺双港区**——东港军用、西港商用
6. **黄金山—白玉山要塞群**——核心制高点
7. **东鸡冠山—二龙山—椅子山堡垒环**——城防外圈
8. **老铁山炮台与海峡警戒线**——海峡咽喉
9. **双岛湾辅港区**
10. **南部沿海卫星农庄群**

**制约辖区发展的三大结构性矛盾**：

1. **人口质量 vs. 速度矛盾**：归化民输入越快，安全审查越稀薄，结社暗流越难控制；输入越慢，劳动力缺口越大，工业产能越受限。
2. **自主 vs. 依附矛盾**：越依赖临高输血（粮食、零件、元老人力），辽南军政委员会政策自主空间越小；越追求自给自足，在达成自给之前的过渡期越危险。
3. **军备 vs. 民生矛盾**：棱堡、炮台、蒸汽军舰每消耗一两，就是民用暖房、粮仓、医院少一块砖——而归化民选择逃跑还是效忠，恰恰取决于那块砖。

## 二、北部防御核心——金州地峡棱堡线（"金州长城"）

旅顺口能否长久立足的关键不在港，而在**金州**。整个辽东半岛南端就像一柄从北向南斜插入海的剑，剑柄的最窄处就是金州地峡——南北宽约3公里，东西被渤海与黄海夹紧。元老院辽南军政委员会的第一项工程就是在这3公里上修筑一条**棱堡防线**，被归化民和大明边民称为"金州长城"。

### 2.1 主棱堡群——"南关岭—大黑山三联堡"

整条防线以三座主棱堡为脊椎，由西至东依次为：

**西堡："渤海堡"**

- 位于渤海湾北侧的高地上，俯瞰金州湾。
- 五角星形棱堡，对角线约260米，墙厚5米，全部使用旅顺铁山水泥厂烧制的硅酸盐水泥+花岗岩外贴+夯土内墙。
- 顶层布置**32磅长管加农炮×4门**（对海岸方向）+ **24磅榴弹炮×6门**（对陆地方向）。
- 棱堡内有独立水井三口、地下弹药库（防爆设计，全混凝土）、储粮库（可供守军3个月）、医疗室、电报站。

**中堡："南关堡"**

- 防线核心，位于金州地峡正中央的南关岭最高点（海拔约87米）。
- 规模最大的复合棱堡，主堡之外有两个外凸的半月堡（demi-lune），形成纵深防御。
- 主堡墙厚6米，对角线320米。
- 主炮台火力：**32磅长管加农炮×6门**、**24磅榴弹炮×8门**、**12磅野战炮×12门**（机动）。
- 内设辽南军政委员会前线指挥所、电报中枢（直通旅顺城区与济州岛）、元老前线居所4套。
- 通过地下隧道与西堡、东堡相连，隧道宽2米、高2.2米，全石砌拱顶。

**东堡："黄海堡"**

- 位于黄海一侧的丘陵上，俯瞰大连湾方向。
- 与西堡同规格的五角星形棱堡。
- 火力配置同西堡，主要炮口指向陆地东北方向，提防清军从复州方向沿海岸南下。

### 2.2 棱堡之间的连接工事

三座主棱堡之间以**连续棱堡墙**相连，总长约3.2公里。墙体特征：

- 墙高约8米，墙厚3米，外侧有宽20米、深6米的干壕（冬季可冻为冰沟）。
- 每隔约200米有一个**菱形小堡角**（demi-bastion）外凸，形成连续交叉火力。
- 墙顶宽4米可通行马车，铺设石板防滑。
- 墙身每隔约40米有一组炮眼+火枪射孔（火枪射孔上中下三层错位布置）。
- 墙外300米设置**鹿砦带**（铁蒺藜+削尖木桩）+ **预警地雷区**（临高制压发雷，密度约每米一枚）。

### 2.3 防线后方设施

- **后勤总仓**：在防线南方约2公里处的"二里桥"村建立后勤总站，含粮仓5座、弹药库3座、马厩200匹、医疗野战站。
- **预备队营盘**：可容纳一个步兵团（约2400人）+ 一个炮兵营（约600人）的常驻兵营。
- **轻便铁路**：从二里桥铁路总站，沿地峡向南有窄轨铁道（轨距762mm，骡马拉动平板车）直通旅顺城区，全长约45公里，用于战时快速运兵和弹药补给。

### 2.4 驻守兵力与日常运作

**防线常态驻守兵力**：

| 位置           | 兵力         | 兵种构成                                   |
| -------------- | ------------ | ------------------------------------------ |
| 渤海堡（西堡） | 380人        | 炮兵连×2 + 步兵连×1 + 工兵排×1             |
| 南关堡（中堡） | 720人        | 炮兵连×3 + 步兵连×2 + 指挥部+通信班+医护排 |
| 黄海堡（东堡） | 380人        | 同渤海堡编制                               |
| 棱堡连接墙     | 480人        | 步兵营×2（分段轮值），含预备冲击队         |
| 二里桥后勤总站 | 600人        | 步兵团预备队一部 + 炮兵营                  |
| **合计**       | **约2560人** | 平时警戒编制；战时可扩充至4500人           |

**日常作息制度**：

- 黎明：全员战备哨位就位，炮手检查炮位、清洁炮管
- 上午：维修轮值（城墙巡视、地雷检查、弹药库盘点）
- 下午：操练轮值（炮手瞄准训练、步兵拼刺操、工兵爆破训练）
- 日落：换岗，夜间哨岗密度加倍
- 每旬：全员实弹演练一次（东堡、西堡各向金州湾方向试射2发）

**补给周期**：

- 轻便铁路每日两班（上午、下午各一）从旅顺城区运至二里桥，转骡马车至各堡
- 弹药储量：各堡保持60天完整备弹（按当时战役烈度可支撑3—5次中等规模进攻）
- 粮食储量：各堡地下粮库储量90天；战时封锁时南关堡可独立坚守最长180天

**冬季特殊规程**：

- 11月起启动"冬季加固"：干壕注水（利用自然结冰制造冰沟障碍，不利于攻城炮车运动）
- 鹿砦带补充铁蒺藜（冻土使木桩拔出更困难，加设金属障碍）
- 兵员补给改为双份御寒物资（棉衣、防冻膏、烈酒定量配给）
- 极寒夜（-15℃以下）：炮位哨兵换班频率从2小时缩短至40分钟，防冻伤

**战略意图**：金州防线的目的**不是全歼清军**，而是"让任何规模的清军进攻都需要付出高昂时间成本"——只要防线能将大规模进攻的推进速度压缩到"一周不超过5公里"，伏波军的蒸汽舰队就有足够时间从临高或济州调兵增援。

## 三、旅顺双港体系

旅顺口的港口分为**东港（军港）**与**西港（商港）**两部分，由黄金山—白玉山山脉天然分隔，仅以一条石板大道与穿山隧道相连。这种分离设计借鉴了旧时空旅顺历史经验——军民分港，便于安保。

### 3.1 旅顺东港（伏波军北方舰队基地）

**港湾规格**：

- 港口入口在黄金山与老虎尾之间，水道宽约300米、深约16米（疏浚后），可通行任何当时代舰艇。
- 入口处由**黄金山炮台**与**老虎尾炮台**形成绝对交叉火力，加上水下铁链（战时升起）封锁。
- 内港水域约2.5平方公里，水深8—14米。

**港内设施**：

- **军用主码头**：石砌深水栈桥4座，每座长250米，可同时停泊8艘千吨级伏波军主力舰，加上中小型舰艇泊位20余处。
- **干船坞×3**：
  - 1号坞：长180米×宽45米，可容纳元老院最大的"威远"级铁肋木壳蒸汽巡洋舰整体入坞大修。
  - 2号坞：长140米×宽30米，主用于中型护卫舰。
  - 3号坞：长90米×宽20米，小型船只与紧急维修。
  - 全部为干船坞，坞门为铁制双开门，蒸汽抽水机抽干海水。
- **海军医院**：独栋三层砖石建筑，60张病床，配备临高出品的麻醉药、抗生素与外科器械。
- **海军学校（北方分校）**：培训归化民水手、炮手、轮机匠，年招生约200人。
- **舰队司令部**："东港统帅楼"，三层红砖建筑，海军元老何鸣的北方驻节地，内有作战室、海图室、电报中枢、信号灯通信塔（与白玉山中继站相连）。

### 3.2 旅顺西港（商港、净化营、对外贸易）

**港湾规格**：

- 利用旅顺西侧的浅水湾改造，水深较浅（4—8米），通过疏浚保证千吨级商船可达。
- 入口较宽，无天然防御，依赖**羊头湾炮台**与海上巡逻。

**港内设施**：

- **商用主码头**：木骨石墩栈桥6座，每座长180—200米，主要停泊归化民商船、登莱商船、对鲜对日贸易船只。
- **海关与商务大厅**：四层青砖+水泥建筑，是元老院在北方最大的关税征收点。
- **净化营**：与屺坶岛模式类似但规模数十倍——可同时处理5000名难民/移民的登记、检疫、分流、培训。
- **大型仓库群**：沿码头纵深500米，红砖仓库40栋，分类为粮食、棉布、药材、武器（出口给朝鲜义军与东江残部）、煤炭、铁器、奢侈品（瓷器、丝绸——对日贸易）。
- **窄轨铁路**：从西港码头延伸至净化营、仓库群、铁山工业区，总长约8公里，使用蒸汽小火车（来自临高的"先驱者"型2-4-0机车）。

**净化营管理流程**（升级版）：

1. **三道闸门入境**：登陆即测量体温、扎种痘、剃头消毒。
2. **8栋联排隔离营房**：每栋容纳500人，砖木结构，连续3—7天医学观察。
3. **筛查与分流**：政保局档案室登记后，按四类分流：
   - A类（青壮男性，健康）→ 工业区、伏波军征召
   - B类（女性、儿童、技工）→ 农庄、纺织厂、学校
   - C类（老弱、待恢复）→ 长期收容所
   - D类（有问题、待审查）→ 隔离审讯区
4. **培训前置教学**：所有人入境后强制完成2周"基础教育"（识字+元老院基础政策）。

### 3.3 双岛湾辅助军港

位于旅顺以西北约15公里的双岛湾，作为东港的备用基地。规模较小，可停泊3—4艘主力舰，主要功能：

- 伏波军近海巡逻艇驻泊地（4艘100—200吨级武装风帆+蒸汽混合动力交通艇）
- 紧急情况下东港封冻时的替代基地
- 雷达观察站（仅元老级别情报，地形上掩盖性较好的瞭望哨）

**西港关税收入估算**：

- 进口税率：粮食2%（战略物资从轻）、奢侈品12%、一般商品5%
- 出口税率：钢铁制品3%、柞蚕绸8%、武器出口特许税20%
- 年关税总收入约**1.2—2.0万两**（详见第二十四章财政体系）

**净化营年度人流**：

- 年均接收难民/移民：约5000—12000人（视辽东战乱烈度波动极大）
- 分流比例：A类（直接可用劳动力）约35%、B类（女性儿童技工）约40%、C类（老弱）约18%、D类（待审）约7%
- 平均通关时间：正常7—14天；疫情期间延长至21—30天

**港口收费体系**：

- 停泊费：东港伏波军舰艇免费；西港商船按吨位收费（每百吨/日0.5两）
- 仓储费：西港仓库每月每间0.8两
- 引港费：外籍船只强制使用元老院引港员（每次2两），防止碰撞或情报泄露

### 3.5 港口季节性运营与冰期应对

| 季节           | 东港状态               | 西港状态                     | 主要挑战                        |
| -------------- | ---------------------- | ---------------------------- | ------------------------------- |
| 春季（3—5月）  | 全速运转               | 贸易旺季开启                 | 开冻后积冰残渣清障              |
| 夏季（6—8月）  | 全速运转               | 贸易最旺（风平浪低）         | 台风预警（每年1—2次需关闭港口） |
| 秋季（9—11月） | 全速运转+备冬补给高峰  | 年度最大商船流量             | 抢在冰期前完成年度储备          |
| 冬季（12—2月） | 主航道不冻，可继续运转 | 浅水区结冰，大型商船限制入港 | 引港员要求高；恶劣天气窗口期窄  |

**关键地理优势**：旅顺东港主航道全年不封冻——这是旅顺在旧时空被沙俄、日本反复争夺的核心原因，也是元老院在黄海—渤海地区拥有唯一全年无冰深水军港的战略意义所在。即便冬季海冰封锁辽东其他港口，伏波军北方舰队依然全年可动。
