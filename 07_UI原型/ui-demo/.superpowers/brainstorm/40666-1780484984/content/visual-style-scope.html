旅顺口 UI 风格修正对照

目标是把 Claude 原型从偏现代 HUD 的质感，拉回早期视频/生成图的冷冬、军政、工业、战地文书语气，同时保留策略经营界面的密度和可读性。

建议修正：主界面 + 共享 UI Kit
REFERENCE

参考风格：冬季战地行政现场

低饱和冷灰、雪地、烟雾、木箱、布帐、铁器和暖灯。画面不是“赛博仪表”，而是历史工业飞地里的军务秩序。

面板应像军务档案、油布地图和铁制铭牌。
强调冷暖对比：大面积冷灰，少量暖橙承担焦点。
减少发光、网格和高饱和青色。
CURRENT

当前问题：策略 HUD 味道过强

布局密度可用，但黑玻璃面板、亮青路线、网格层和大号数字让视觉重心从“旅顺口历史世界”偏到泛科幻/现代数据看板。

背景图被压暗，失去视频样片的雪地空气感。
资源栏和按钮的颜色过于游戏化。
节点图标更像抽象 HUD，不像地图上的军务标注。
PROPOSED

拟定方向：冷铁军务界面

保留三栏管理布局，改为低透明深铁面板、羊皮纸色文字、砖红警报、钝金路线、少量海军蓝/钢蓝辅助。界面像压在战役地图上的军政调度台。

去掉全屏科技网格，换成轻微暗角和纸面纹理。
路线和节点降低亮度，改成地图标注与军令旗色。
按钮与状态条统一成铁、纸、蜡封、灯火的材质语气。

推荐范围

先修 `index.html` 的主界面视觉，并同步 `ui-kit.css` 的变量、基础面板、按钮、状态组件。这样首屏和其它页面的基础色彩会统一，改动范围可控。

不建议本轮做

不逐页重画 `strategic-map`、`tech-tree` 等功能页的专属布局，除非你希望这次把所有页面都按同一风格做完整视觉翻新。
